# Task 3: Model Deployment Project

## Project Title

Machine Learning Model Deployment using Flask

## Objective

The objective of this project is to deploy a machine learning model as a web application using Flask.

## Tools and Technologies Used

* Python 3.x
* Flask
* Scikit-learn
* Jupyter Notebook

## Project Workflow

### 1. Model Preparation

The trained machine learning model was saved for deployment.

### 2. Flask Application

A Flask web application was created to serve predictions.

### 3. API Development

A route was developed to receive input and return predictions.

### 4. Testing

The application was tested locally using a web browser.

## Application Features

* Loads trained model
* Accepts user input
* Generates predictions
* Displays output through a web interface

## Project Structure

app.py
requirements.txt
README.md
saved_model.pkl

## How to Run

1. Install dependencies:

pip install flask scikit-learn

2. Start the application:

python app.py

3. Open browser:

http://127.0.0.1:5000

## Expected Output

The application runs successfully and returns predictions generated by the trained machine learning model.

## Conclusion

This project demonstrates the deployment of a machine learning model using Flask. The deployed application allows users to interact with the model through a simple web interface.
